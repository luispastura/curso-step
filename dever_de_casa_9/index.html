Pseudo-classes and pseudo-elements

O próximo conjunto de seletores que veremos é chamado de pseudoclasses e pseudoelementos. Há um grande número deles e geralmente servem a propósitos bastante específicos. Depois de saber como usá-los, você pode consultar a lista para ver se há algo que funcione para a tarefa que está tentando realizar. Mais uma vez, a página MDN relevante para cada seletor é útil para explicar o suporte do navegador.

What is a pseudo-class?

Uma pseudoclasse é um seletor que seleciona elementos que estão em um estado específico, por exemplo, eles são o primeiro elemento de seu tipo ou estão sendo direcionados pelo ponteiro do mouse. Eles tendem a agir como se você tivesse aplicado uma classe a alguma parte do seu documento, muitas vezes ajudando a reduzir o excesso de classes em sua marcação e fornecendo um código mais flexível e de fácil manutenção.

User-action pseudo classes

  • Hover — mentioned above; this only applies if the user moves their pointer over an element, typically a link.
  • Focus — only applies if the user focuses the element by clicking or using keyboard controls.

What is a pseudo-element?

Pseudo-elementos se comportam de maneira semelhante. No entanto, eles agem como se você tivesse adicionado um novo elemento HTML à marcação, em vez de aplicar uma classe aos elementos existentes. Pseudo-elementos começam com dois pontos duplos ::. ::before é um exemplo de pseudo-elemento.
